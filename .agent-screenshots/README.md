# 자소서 글자수 체크 앱 - 스크린샷 안내

## 접속 방법
브라우저에서 다음 URL로 접속하세요:
http://localhost:3001

## 주요 화면

### 1. 메인 화면
- 상단: "📝 자소서 글자수 체크" 제목
- 통계 요약: 총 파일 개수 3개, 총 글자수 표시
- 파일 목록: 3개의 카드 (자기소개서1.txt, 지원동기.txt, 경력사항.txt)
- 각 카드에는 파일 이름과 글자수가 표시됨

### 2. 모달 화면 (파일 카드 클릭 시)
- 파일 전체 내용 표시
- 상단: 파일 이름 (인디고-퍼플 그라데이션 배경)
- 중간: 파일 전체 텍스트
- 하단: 글자수 정보

## 테스트 방법
1. 메인 화면에서 파일 목록 확인
2. 아무 카드나 클릭
3. 모달 팝업 확인
4. ESC 키 또는 X 버튼으로 모달 닫기
5. 모달 바깥 영역 클릭해도 닫힘

## 파일 구조
- index.html: HTML 구조 (Tailwind CSS 사용)
- client.js: JavaScript 로직 (파일 읽기, 글자수 계산)
- profile/: 자소서 파일들이 저장된 폴더
