【개인화 추천 시스템 고도화 프로젝트】
기간: 2023.06 ~ 2024.01 (8개월)
역할: 데이터 분석 리드

프로젝트 개요:
기존의 단순한 인기도 기반 추천에서 개인화된 추천 시스템으로 전환하여 사용자 경험과 매출을 개선하는 프로젝트를 수행했습니다. 협업 필터링과 컨텐츠 기반 필터링을 결합한 하이브리드 추천 시스템을 구축했습니다.

담당 업무:
- 사용자 행동 데이터 수집 및 전처리 파이프라인 구축
- EDA(탐색적 데이터 분석)를 통한 추천 알고리즘 선정
- Matrix Factorization과 Deep Learning 기반 추천 모델 개발
- A/B 테스트 설계 및 결과 분석

기술적 도전과 해결:
Cold Start 문제를 해결하기 위해 신규 사용자와 신규 상품에 대해 콘텐츠 기반 필터링을 적용했습니다. 또한 실시간 추천을 위해 모델 서빙 인프라를 최적화하여 응답 시간을 200ms 이하로 유지했습니다.

성과:
- 추천 시스템 CTR 2.5%에서 4.1%로 64% 향상
- 추천 상품 구매 전환율 30% 증가
- 사용자 체류 시간 평균 15% 증가
- 모델 성능 지표(NDCG) 0.65에서 0.82로 개선

8. 팀워크 경험
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

이전 프로젝트에서 마케팅 팀과 데이터 분석 팀 간의 커뮤니케이션 문제로 분석 요청이 명확하지 않아 여러 번 재작업이 발생했습니다. 마케팅 팀은 필요한 인사이트를 명확히 전달하지 못했고, 분석 팀은 비즈니스 맥락을 충분히 이해하지 못한 채 분석을 수행했습니다.

이를 해결하기 위해 분석 요청 템플릿을 만들었습니다. 비즈니스 목표, 의사결정 시나리오, 필요한 인사이트 등을 구체적으로 작성하도록 했고, 분석 시작 전 킥오프 미팅을 통해 요구사항을 명확히 하는 프로세스를 도입했습니다.

또한 정기적으로 데이터 리터러시 세션을 진행하여 마케팅 팀이 데이터를 이해하고 활용하는 능력을 향상시켰습니다. 기본적인 SQL 쿼리 작성법과 대시보드 활용법을 교육했습니다.

결과적으로 분석 요청의 명확성이 향상되어 재작업이 80% 감소했고, 마케팅 팀이 직접 간단한 분석을 수행할 수 있게 되어 분석팀의 업무 부담도 줄었습니다.

9. 문제해결 경험
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

매출 예측 모델의 정확도가 급격히 떨어지는 문제가 발생했습니다. 이전에는 MAPE(평균 절대 백분율 오차) 5% 이내의 정확도를 보이던 모델이 갑자기 15% 이상의 오차를 보이기 시작했고, 이로 인해 재고 관리와 마케팅 예산 배분에 문제가 발생했습니다.

문제 분석:
데이터 분포를 확인한 결과, COVID-19로 인한 소비 패턴의 급격한 변화가 원인이었습니다. 기존 모델은 과거 데이터만을 기반으로 학습되어 새로운 패턴을 반영하지 못하고 있었습니다.

해결 과정:
1. 최근 3개월 데이터에 더 높은 가중치를 부여하는 시간 가중 학습을 적용했습니다.
2. 외부 변수(코로나 확진자 수, 재택근무 비율 등)를 추가 피처로 포함했습니다.
3. 앙상블 모델을 구축하여 여러 모델의 예측을 결합했습니다.
4. 모델 성능을 실시간으로 모니터링하고, 정확도가 떨어지면 자동으로 재학습되는 시스템을 구축했습니다.

결과:
- 예측 정확도가 MAPE 6%로 회복
- 재고 부족 및 과잉 재고 문제 40% 감소
- 마케팅 ROI 25% 향상
- 자동 재학습 시스템으로 지속적인 모델 성능 유지

이 경험을 통해 데이터와 비즈니스 환경의 변화를 지속적으로 모니터링하고, 모델을 업데이트하는 것의 중요성을 배웠습니다.

10. 리더십 경험
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

작년에 데이터 분석 조직의 역량 강화 프로젝트를 리드했습니다. 3명의 주니어 분석가가 합류하면서 팀의 역량을 균일하게 높이고, 효율적인 협업 체계를 구축할 필요가 있었습니다.

리더십 발휘 사례:

1. 지식 공유 문화 조성

2. 표준화 및 문서화
